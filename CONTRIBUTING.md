# Welcome to Open Source – Contributor Guide

Thank you for your interest in contributing! 🎉  
This repository is designed to help beginners make their **first open-source contribution**.

Please follow the steps below **carefully** to ensure your Pull Request (PR) is accepted.

---

## 📌 Who Can Contribute?

- Beginners to Open Source
- Students
- Anyone learning Git & GitHub

No prior experience is required.

---

## 🚀 Step-by-Step Contribution Guide

### 1️⃣ Fork the Repository
- Click the **Fork** button on the top-right of this repository.
- This creates a copy of the repository under your GitHub account.

---

### 2️⃣ Clone Your Fork
